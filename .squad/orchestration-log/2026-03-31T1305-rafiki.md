## Key Decisions

1. **CSS overlay pattern:** RTL implemented as ~60-line additive CSS layer rather than duplicating 300+ lines of base styles. Maintains DRY principle; future style fixes apply to both RTL and LTR automatically.

2. **Font stack:** `'Segoe UI', 'Arial Hebrew', 'Noto Sans Hebrew', Arial, sans-serif` — bundled with Chromium (Playwright's engine), no external font installation needed.

3. **Code stays LTR:** Using `unicode-bidi: bidi-override` on code blocks and `unicode-bidi: embed` on URLs ensures technical content renders left-to-right regardless of document direction.

4. **Line-height tuning:** Hebrew characters benefit from extra vertical space:
   - Normal mode: 1.8 (vs 1.7 base)
   - Compressed mode: 1.45 (vs 1.35 base)
   - Proportional increase in both cases

5. **Language detection:** `isRtlLanguage()` supports he, ar, fa, ur — easily extensible for future RTL languages.
